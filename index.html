🚀 What is p6s?

p6s is a specialized terminal tool that makes managing PostgreSQL databases in Kubernetes clusters incredibly simple. No more complex kubectl commands, port-forwarding hassles, or hunting for connection credentials - p6s automatically discovers your PostgreSQL pods, extracts connection details from secrets, and provides instant access through an intuitive terminal interface.

☸️ One-Click Kubernetes PostgreSQL Access

  • Auto-Discovery: Instantly find all PostgreSQL pods across namespaces
  • Zero Configuration: Automatic credential extraction from Kubernetes secrets
  • Smart Connection: Seamless port-forwarding and connection management
  • Multi-Cluster Support: Switch between different Kubernetes contexts effortlessly

🔍 Real-time Monitoring

  • Live connection status and performance metrics
  • Database table statistics and health monitoring

⚡ Developer Friendly

  • Terminal-based interface - no GUI required
  • Custom SQL queries and command execution

🛠️ Installation

Building from Source

git clone https://github.com/fanxiangyu/p6s.git
cd p6s
make build

Requirements

  • Go 1.19 or later
  • PostgreSQL client libraries
  • kubectl (for Kubernetes integration)
